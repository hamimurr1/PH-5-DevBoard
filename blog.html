Question-1: What are the different ways to select an element in the DOM?

We can select an element in DOM many different ways like:

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()
  • getElementsByName()
Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML:

Returns or sets everything, including the HTML code inside the element.

innerText:

Returns or sets the visible text of the element.
Question-3: What is event delegation in the DOM?
Event delegation is a technique where events on child elements are handled by adding event handlers to the parent element.
Question-4: What is event bubbling in the DOM?
Event bubbling is a process of processing events in the DOM, where the event first occurs on a specific target element and then gradually progresses up to its parent elements.
Question-5: How do you create, add, and remove elements using JavaScript?
We use some function for create, add, and remove.They are:
  • document.createElement("create element")
  • append("add")
  • appendChild("add")
  • prepend("add")
  • insertBefore("add")
  • remove("remove")
  • removeChild("remove")
